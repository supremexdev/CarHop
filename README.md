
  # Car Rental Web App

  This is a code bundle for Car Rental Web App. The original project is available at https://www.figma.com/design/umETEyUCe1kPYem2Dy5WS5/Car-Rental-Web-App.

  ## Running the code

  navigate to folder containing server.js 
    in the command line run 'npm run server'

  in the same folder
    Run `npm i` or 'npm install' to install the node.js dependency folders.

    Run `npm run dev` to start the development server.
